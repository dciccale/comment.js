$B.ui.dialog.create(options) # Defined in: bbva.ui.dialog.js:759

Creates a ui dialog with given options

Arguments

options
object
object of name/value pairs
  • {
    • el string optional selector for the element to act as a dialog
    • id string unique id for the dialog
    • content string string content for de dialog (could be the inner html of a div $('#myDiv').html())
    • afterLoad callback callback function to execute after loading dialog content via ajax
    • button objectstring jQuery object or selector for the button that triggers the dialog
    • url stringfunction url string or function that returns a url string
    • urlParams object, function object or function that returns an object with params for the url
    • tagName string the tagName for creating the dialog if no element was provided (div by default)
    • dialogOptions object Any jQuery ui option (see jquery ui documentation)
    • onCreate function callback that fires only once when the dialog is created
    }

Returns object jQuery object of the dialog content

Usage

Another text yoh!

 $B.ui.dialog.create({
  id: 'MyDialog',
  el: '#DialogElement',
  button: '#button',
  dialogOptions: {
    draggable: true,
    width: 640
  }
 });
