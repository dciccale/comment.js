$B.ui.dialog.create(options)#Defined in: bbva.ui.dialog.js:749

Creates a ui dialog with given options

Arguments

options
object
object of name/value pairs
  • {
    • elstring optionalselector for the element to act as a dialog
    • idstringunique id for the dialog
    • contentstringstring content for de dialog (could be the inner html of a div $('#myDiv').html())
    • afterLoadcallbackcallback function to execute after loading dialog content via ajax
    • buttonobject stringjQuery object or selector for the button that triggers the dialog
    • urlstring, functionurl string or function that returns a url string
    • urlParamsobject, functionobject or function that returns an object with params for the url
    • tagNamestringthe tagName for creating the dialog if no element was provided (div by default)
    • dialogOptionsobjectAny jQuery ui option (see jquery ui documentation)
    • onCreatefunctioncallback that fires only once when the dialog is created
  • }

Returns object jQuery object of the dialog content

Usage

$B.ui.dialog.create({
  id: 'MyDialog',
  el: '#DialogElement',
  button: '#button',
  dialogOptions: {
    draggable: true,
    width: 640
  }
 });
